To diagnose the crash, please do the following:

1. In Android Studio, open Logcat
2. Filter by your package name: com.fariha.deshistore
3. Look for lines with 'AndroidRuntime', 'FATAL EXCEPTION', or 'java.lang.NullPointerException'
4. Share those error lines here

The most likely issues are:
- Missing Firestore security rules in Firebase Console
- Missing google-services.json configuration
- App not rebuilt after code changes

Quick Fix to Try:
===================
1. Go to Firebase Console > Firestore Database > Rules tab
2. Replace rules with:
   rules_version = '2';
   service cloud.firestore {
     match /databases/{database}/documents {
       match /{document=**} {
         allow read, write: if true;
       }
     }
   }
3. Click 'Publish'
4. In Android Studio: Build > Clean Project
5. Build > Rebuild Project
6. Run the app again
